Run it locally

The preview above is the real UI. To run the live tools, clone the repo:

git clone https://github.com/Paius-George/osint-station.git
cd osint-station
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python app.py

Then open http://127.0.0.1:5001/. Full details in the README.
